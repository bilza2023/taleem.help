

 Number Line and Ordering Numbers

آج ہم **number line** کو تفصیل سے سمجھیں گے،
کیونکہ number line صرف ایک لکیر نہیں ہے،
بلکہ یہ نمبروں کو *سمجھنے کا سب سے مضبوط ذریعہ* ہے۔

سب سے پہلے، اس تصویر کو غور سے دیکھیں۔
یہ ایک سیدھی افقی لائن ہے،
جس پر مختلف نمبرز برابر فاصلے پر نشان کی صورت میں لگے ہوئے ہیں۔
اسی لائن کو number line کہا جاتا ہے۔

number line کا سب سے اہم نقطہ
اس کے بالکل درمیان میں موجود **صفر** ہے۔
صفر ایک عام نمبر نہیں ہے،
بلکہ یہ ایک **reference point** ہے۔
یعنی ہم تمام نمبروں کو صفر کے مقابلے میں سمجھتے ہیں۔

اب صفر کے دائیں طرف دیکھیں۔
یہاں ہمیں 1، 2، 3، 4، 5 جیسے نمبرز نظر آتے ہیں۔
یہ سب **positive numbers** ہیں۔
جیسے جیسے ہم دائیں طرف آگے بڑھتے ہیں،
نمبروں کی value بڑھتی جاتی ہے۔
اس کا مطلب یہ ہے کہ
number line پر دائیں جانب جانا
ہمیشہ *زیادہ* کی طرف جانا ہوتا ہے۔

اب صفر کے بائیں طرف آئیں۔
یہاں ہمیں −1، −2، −3، −4 جیسے نمبرز نظر آتے ہیں۔
یہ **negative numbers** کہلاتے ہیں۔
اور جیسے جیسے ہم بائیں طرف جاتے ہیں،
نمبروں کی value کم ہوتی جاتی ہے۔

یہاں ایک بہت اہم بات سمجھنے والی ہے۔
اگرچہ −5 لکھنے میں بڑا لگتا ہے،
لیکن number line پر −5، −1 سے زیادہ بائیں طرف ہے،
اس لیے −5 **چھوٹا** نمبر ہے۔
number line ہمیں یہی غلط فہمی دور کرنے میں مدد دیتی ہے۔

اب number line کا ایک بنیادی اصول یاد رکھیں:
**جو نمبر دائیں طرف ہوتا ہے، وہ بڑا ہوتا ہے۔**
اور
**جو نمبر بائیں طرف ہوتا ہے، وہ چھوٹا ہوتا ہے۔**

مثال کے طور پر،
اگر ہم −2 اور 3 کا موازنہ کریں،
تو 3 صفر کے دائیں طرف ہے
اور −2 بائیں طرف،
اس لیے 3، −2 سے بڑا ہے۔

اسی طرح،
اگر ہم −1 اور −4 کا موازنہ کریں،
تو −1 دائیں طرف ہے،
اس لیے −1، −4 سے بڑا ہے،
حالانکہ دونوں negative ہیں۔

اب ان تیروں پر غور کریں
جو number line کے دونوں سروں پر لگے ہوئے ہیں۔
یہ تیر ہمیں یہ بتاتے ہیں
کہ number line ختم نہیں ہوتی۔
نمبرز دونوں طرف
ہمیشہ آگے بڑھتے رہتے ہیں۔

number line کا استعمال صرف نمبرز دکھانے کے لیے نہیں،
بلکہ ordering اور comparison کے لیے ہوتا ہے۔
جب ہم نمبرز کو چھوٹے سے بڑے کی ترتیب میں لکھتے ہیں،
یا یہ فیصلہ کرتے ہیں کہ کون سا نمبر بڑا ہے،
تو ہم دراصل ذہنی طور پر number line استعمال کر رہے ہوتے ہیں۔

اگر آپ number line کو صحیح طرح سمجھ لیں،
تو integers، fractions، decimals،
اور آگے چل کر algebra
سب زیادہ واضح اور آسان ہو جاتا ہے۔

اسی لیے کہا جاتا ہے کہ
number line
میَتھ کی زبان سمجھنے کی چابی ہے۔
