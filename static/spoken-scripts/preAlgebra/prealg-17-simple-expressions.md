
**Lesson 17: Simple Expressions (No Solving Yet)**

آج ہم **Simple Expressions** کو سمجھیں گے۔
ہم ابھی solve نہیں کریں گے،
صرف یہ دیکھیں گے کہ expression اصل میں ہوتا کیا ہے۔

Screen کے اوپر لکھا ہے: **Simple Expressions**۔
Expression کا مطلب ہے
numbers اور letters کو ساتھ رکھ کر لکھنا۔

اب تصویر کے پہلے حصے کو دیکھیں،
جہاں لکھا ہے: **Parts together**۔

یہاں چھوٹے orange boxes دکھائے گئے ہیں۔
کچھ boxes پر **a** لکھا ہوا ہے،
اور کچھ boxes numbers کو show کر رہے ہیں۔

ہر box ایک part ہے۔
جب ہم ان parts کو ساتھ رکھتے ہیں،
تو ایک expression بنتا ہے۔

اب اگلے حصے کی طرف آئیں،
جہاں لکھا ہے: **Expression as a whole**۔

یہاں سارے boxes
ایک bracket کے اندر آ گئے ہیں۔

اس کا مطلب یہ ہے کہ
اب ہم ان سب کو
ایک ہی expression کے طور پر دیکھ رہے ہیں۔

یہ پورا group
ایک complete expression ہے۔

اب آخری حصے کو دیکھیں،
جہاں لکھا ہے: **Meaning**۔

یہ حصہ بتا رہا ہے کہ
یہ expression کسی value کے لیے stand کرتا ہے۔

یہاں **a** کوئی specific number نہیں ہے۔
یہ ایک variable ہے،
جو کسی بھی value کو represent کر سکتا ہے۔

تو جب ہم کہتے ہیں
کہ یہ ایک expression ہے،
تو اس کا مطلب ہوتا ہے:
numbers اور variables کا ایک meaningful combination۔

یاد رکھیں:

Expression کوئی question نہیں ہوتا۔
Expression کا answer نہیں نکالا جاتا۔
Expression صرف structure دکھاتا ہے۔

Solving بعد میں آئے گی۔
ابھی ہم صرف **understanding** بنا رہے ہیں۔

اور یہی
Simple Expressions کا basic idea ہے۔ ✅
